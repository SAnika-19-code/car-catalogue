# Varsha Cushions — Premium Car Accessories Catalogue Platform

Developed a modern full-stack catalogue platform for a real automobile accessories business using Next.js, Firebase Firestore, Cloudinary, and Vercel.

The project includes a professional customer-facing gallery system and a fully functional admin dashboard for managing company catalogues, image uploads, image organization, and product presentation.

Key features include:

• Responsive masonry-style image gallery
• Fullscreen image viewer with keyboard navigation
• Admin authentication system
• Multi-image upload support using Cloudinary CDN
• Drag-and-drop image reordering
• Company-based catalogue architecture
• Bulk image selection and deletion
• Recycle bin system for deleted content
• Real-time Firestore database integration
• Mobile-optimized responsive UI
• Vercel deployment with environment variable management
• Optimized image delivery using Cloudinary transformations

The platform was designed with a premium dark luxury aesthetic using a custom color palette and modern responsive design principles.

Tech Stack:
Next.js • React • TypeScript • Firebase Firestore • Cloudinary • Tailwind CSS • Vercel
